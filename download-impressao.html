Servidor de Impressão

MagaPet — Daruma DR800

servidor_impressao.py Servidor principal
requirements_impressao.txt Dependências Python
iniciar.bat Iniciar o servidor
⚠️ Salve os 3 arquivos na mesma pasta: C:\MagaPet\impressao\
Após baixar os 3 arquivos:
1. Aperta Win + R, digita cmd, aperta Ctrl+Shift+Enter
2. Digita: cd C:\MagaPet\impressao → Enter
3. Digita: pip install -r requirements_impressao.txt → Enter
4. Aguarda instalar e clica duas vezes em iniciar.bat
